# ezTensorflowService
A serverless service for Tensorflow.js with AWS Lambda

### Key Features
* Serverless Framework
* Nodejs backend
* Tensorflow.js
* Typescript
* AWSLambda

### Prerequisite dependencies
1. serverless
2. typescript
3. ts-node
